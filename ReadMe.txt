Project: CSS Practice Collection
Description:
A collection of small CSS projects and exercises demonstrating core CSS concepts: the box model, selectors, layout techniques (flexbox, grid), animations, responsive design, and more.

How to view:
- Open any folder and double-click its `index.html` file, or right-click and "Open with Live Server" in VS Code for live reload.
- Recommended browsers: Chrome, Firefox, Edge (latest versions).

Folder  structure:
- assignment/               - basic practice project
- box model/               - box model examples
- css selectors/           - selector exercises
- css variables/           - using CSS custom properties
- display/                 - display property demos
- flexbox/                 - flexbox layouts and examples
- grid/                    - CSS Grid layouts
- grid-portfolio/          - portfolio built with grid
- keyframe animations/     - @keyframes & animations
- media queries/           - responsive design examples
- overflow/                - overflow behavior demos
- positions/               - positioning (static, relative, absolute, fixed)
- texts and fonts/         - typography and font usage
- transform/               - transform examples (rotate, scale, translate)
- transition/              - transitions & timing functions
- units/                   - length units and comparisons.

Conventions & notes:
- Each folder contains an `index.html` and `style.css`. Open the HTML file to see the demo.
- Keep CSS files modular & commented for clarity.
- Use the browser DevTools to inspect styles and experiment.

Contributing:
- Add new small demos in a new folder with `index.html` and `style.css`.
- Keep examples small, focused, and well-commented.
